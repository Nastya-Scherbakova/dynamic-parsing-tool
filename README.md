## Description

Parsing tool that can return object which scheme you provided in request using url of sites that you need (*POST* `/creator`). Was created using [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
